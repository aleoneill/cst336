FAQ's

I'm new to programming, which programming language should I learn?

According to Stackify, the top 5 most popular programming languages are:

  1. Java
  2. The C Programming Language (C++, C#, C)
  3. Python
  4. Javascript
  5. Ruby

Some intro to programming classes use C++, but any of the top 3 most common languages would be a great start!

What programming language should I use if I want to create an app?

Depends on the platform you want it to run on. If you want to create a software based app (Calculator, Calendar), you can use languages like C++ (which can be run through your terminal shell) or Python. From there you can connect it to HTML and CSS and make a web application. If you want to create an iOS app, you'll need to use Swift (which requires MacOS - Swift runs through XCode which is only available for MacOS). If you want to create an Android app, you'll want to use Java (can use any OS, you'll just need to download Android Studio).

Is it easy to remember different languages?

Yes and no. Generally different languages have similar syntax (except for Python. Python is almost like speaking English into the computer). Check out the code snippets below to find out how to print in different languages:

Swift

let name = "Chris" print("Hello, \(name)!") // should print "Hello Chris!"

C++

cout << "Hello World!" << endl;

Python

print("Hello World!")
