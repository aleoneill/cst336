40 years ago, a Ph.D student (Bjarne Stroustrup) messed around with C and created a language that is found in operating systems, web browsers, microcontrollers, videogames, and anywhere else!

Features:

  • Compiled language: compiles directly to a machine's native code, allowing it to be one of the fastest languages in the world (if optimized properly).

  • Strongly-typed unsafe language: expects the programmer to know what they're doing but allows for incredible amounts of control as a result.

  • Supports static and dynamic type checking: allows type conversions to be checked either at compile-time or run-time, offering flexibility. However most C++ type checking is static.

  • Portable: C++ has a wide range of compilers that run on many different platforms that support it. Code that exclusively uses C++'s standard library will run on many platforms with little to no changes.

  • Upwards compatible with C: C++ is built off of C, meaning it's compatible with almost all C code. C++ can use C libraries with little ot no modifications of the libraries' code.
