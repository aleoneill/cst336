As Apple calls it..

The powerful programming language that is also easy to learn.

Features:

  • To best support international languages and emoji, Strings are Unicode-correct and use a UTF-8 based encoding to optimize performance for a wide-variety of use cases.

  • Variables are always initialized before use, arrays and integers are checked for overflow, memory is automatically managed, and enforcement of exclusive access to memory guards against many programming mistakes.

  • Swift heavily leverages value types, especially for commonly used types like Arrays and Dictionaries.

  • Swift is a successor to both the C and Objective-C languages.

  • Includes low-level primitives such as types, flow control, and operators.

  • It also provides object-oriented features such as classes, protocols, and generics.

  • Designed to be anyone’s first programming language, whether you’re still in school or exploring new career paths.
