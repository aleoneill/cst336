Python is an interpreted, high level, general-purpose programming language created by Guido van Rossum in 1991.

Python emphasizes code readability with its notable use of significant whitespace. Constructs an OOP approach to help programmers write clear, logcial code for small and large-scale projects.

Features:

  • Easy to learn and use: developer friendly and high leveled

  • Expressive: more understandable and readable

  • Interpreted language: Reads line by line at a time making debugging more simple.

  • Cross-platform language (Portable): runs on MacOS, Windows, Linux, etc.

  • OOP Language: supports object oriented language and concepts of classes and objects come into existence

  • Extensible: implies other languages can be used to compile the code

  • Large standard library: provides rich set of module and functions for app development

  • Integrated: can be easily integrated with languages like C, C++, Java, etc.
